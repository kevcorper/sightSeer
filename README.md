# sightSeer
Sight-seeing app using location and ARKit to display close-by areas of interest in augmented reality.

## Setup
App uses ARKit and camera, so a real iPhone device (6s or above) is necessary

## Tech
1. Swift 4
2. iOS 11
3. Xcode 9
4. ARKit
5. CoreLocation
6. GameplayKit
